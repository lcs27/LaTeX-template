\subsection{Template for article}
3 predefined templates are given in this template:
\begin{itemize}
    \item LCSArticle, which is the basic template
    \item CSArticle, which is the template based on LCSArticle for CentraleSup√©lec
    \item BHArticle, which is the template based on LCSArticle for Beihang University.
\end{itemize}\par
The following code gives how this document is programmed
\begin{lstlisting}[style=TeX]
\documentclass[english]{LCSArticle}% english can be changed to french
\title{\LaTeX \ Template by LCS}
\author{$lcs_{27}$}

\begin{document}
\MakeContentTitle % \MakeSimpleTitle or \MakeComplateTitle are also usable
\end{document}
\end{lstlisting}\par
Up to know, only 2 languages(English, French) are available.

\section{Required Packages}
Several packages are preloded in this template, the following code shows how they are loaded in the template:
\begin{lstlisting}[style=TeX]
%% Language Configuration
\RequirePackage[utf8]{inputenc}%For French accents
\RequirePackage[T1]{fontenc}%For French babel
\RequirePackage{ifthen}

%% Page
\RequirePackage{geometry}
\geometry{a4paper,left=2.54cm,right=2.54cm,top=3.18cm,bottom=3.18cm}
\RequirePackage{titlesec}   %Margin

%% Bibliography & Reference
\RequirePackage{natbib} %Bibliography
\RequirePackage{hyperref} %Reference
\RequirePackage{url} % Use url as reference

%% Math
\RequirePackage{amsmath}%Math formula
\RequirePackage{amsfonts}%Math fonts
\RequirePackage{amsthm}%Theorem-like structures
\RequirePackage{amssymb}%Math symbols
\RequirePackage{mathrsfs}%Math fonts
\RequirePackage{mathtools} %Math symbols 
\RequirePackage{siunitx} %Notation scientific (Ex.: \num{2e+9})

%% Graph & Color
\RequirePackage{graphicx} %use graph format
\RequirePackage[justification=centering]{caption} %Graph caption
\RequirePackage{subcaption} %Graph sub-caption
\RequirePackage{float} %Graph Placement
\RequirePackage{epstopdf} %Convert EPS to PDF
\RequirePackage{xcolor} %Colors


%% Table
\RequirePackage{diagbox} %Diagonal in tables
\RequirePackage{multirow} %Multirows

%% Programming
\RequirePackage{listings} %Programming
\end{lstlisting}
